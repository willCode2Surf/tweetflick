# build README

 - The build process can be done by executing build.xml
 - The process is based on the wink build
 - modules.json and profiles.json contains the definition of CSS and JS components
 - build.properties contains paths and build parameters
 
It may be required to change the following properties :

 - tf.wink.relativedir : the relative path to wink from project
 - tf.project.relativedir : the relative path to project from wink
